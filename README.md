# 梦境牌国 - 持牌人原型

Vite 静态网页原型，构建产物为纯静态文件，可部署到 GitHub Pages。

本地运行：

```bash
npm install
npm run dev
```

然后打开：

```txt
http://127.0.0.1:5173/
```

构建静态产物：

```bash
npm run build
```

推荐演示路径：

1. 在开始页直接使用默认 Seed，点击“开始一局”。
2. 在地图页选择可抵达战斗节点，查看节点确认弹窗和奖励说明。
3. 进入战斗后点击“选择建议”，观察牌型预览、行动顺序、主目标、保留牌和上一手牌摘要。
4. 战斗胜利后进入结算页，确认卡包、Joker、经验和升级奖励。
5. 回到地图后打开背包、卡册、卡包管理或商店，展示局外构筑和经济系统。
6. 在实验室页复制 Seed / 存档，或运行固定策略 smoke 试跑。

当前实现目标：

- 数据化配置：`src/data.js` 管理卡牌、藏品、怪物、Boss、事件池
- 12 位显示 seed + 字符串 hash 随机系统
- 26 张卡册、7 张手牌、首牌保留
- 调度点重抽
- 德州扑克 5 张牌型判定
- 当前牌型预览、建议牌型、一键选择建议
- 14 个普通怪、9 个精英怪、6 个 Boss 与简单最优意图
- 经验、Joker、卡包、升级
- 简化地图推进
- 商店刷新、藏品栏位、卡包栏位
- 大背包详情、26 张卡册编辑
- 浏览器本地存档
- 71 个规则书藏品与主要效果接入
- 基础事件池、精英/Boss 奖励、层级卡包概率
- 实验室页：seed/RNG 状态、数据池统计、存档导出、固定策略 smoke 试跑
- PC 端 UI：开始页、地图、战斗、结算、商店、背包、卡册、卡包管理、通关/失败页
- 第一版轻量反馈：选牌、判定区入场、战斗摘要、血条过渡、日志高亮
